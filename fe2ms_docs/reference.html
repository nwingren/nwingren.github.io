API reference

Main public interface

The following classes make up the core of the public part of the code. These are callable directly from the top module for convenience, but the source code is located in the systems and utility submodules.

fe2ms.ComputationVolume(mesh_file, ...[, ...])

Mesh, material properties and boundary conditions for a FEBI computation volume.

fe2ms.FEBISystemFull(frequency, ...[, ...])

FEBI system using the full block matrix structure.

fe2ms.FEBISystemACA(frequency, ...[, ...])

FEBI system where the BI blocks are assembled using the adaptive cross approximation (ACA).

The following submodules have useful contents to the classes above.

fe2ms.preconditioners

Various preconditioners used for iterative solution of full FE-BI system.

fe2ms.materials

Contains electromagnetic properties of various materials on the format (epsr, mur) where each property can be complex.

Internal submodules

Submodules which are mostly internal and rarely need to be called directly.

fe2ms.systems

Finite element boundary integral systems based on different solution approaches.

fe2ms.result_computations

Internal functions for computing results with numba JIT compilation.

fe2ms.assembly

Functions for assembly of the FE and BI blocks.

fe2ms.assembly_nonsingular_full

Internal assembly functions for full nonsingular matrix assembly with numba JIT compilation.

fe2ms.assembly_nonsingular_aca

Internal assembly functions for ACA nonsingular matrix assembly with numba JIT compilation.

fe2ms.bi_space

BI mesh and basis data.

fe2ms.utility

Utility classes and internal functions.
